# Installing

### 1. 安装Rust和系统依赖项（MacOS）
1. 安装 CLang 和 macOS 
`xcode-select --install`

2. 在 macOS 上安装 Rust
`curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`

3. 检查是否正确安装了 Rust
`rustc --version`

### 2. 安装 Tauri CLI
1. `cargo install tauri-cli --version "^1.0.0"`
2. `cargo tauri --help`

### 3. 安装 Angular CLI

# Run

1. `npm run tauri dev`
